ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ


Проект Examiner предназначен для прохождения экзаменов учеником в режиме online, а также для подготовки их учителем.
Examiner содержит 3 модуля: student, teacher и server.

Модуль student (student.exe) необходим для ученика, который хочет пройти экзамен.
Модуль teacher (teacher.exe) предназначен для учителя, который может создавать, просматривать и редактировать экзамены.
Модуль server (server.exe) - это непосредственно сам сервер, использующий базу данных, и необходимый для запуска модулей student и teacher.

1) Запустите файл server.exe (это необходимо для дальнейшей работы с проектом).
2) Запустите файл student.exe или teacher.exe.
3) Зайдите в настройки (они находятся в левом нижнем углу), и нажмите "Проверить соединение".
4) Если соединение с сервером не удалось, то введите в поле выше текущий IP-адрес сервера и повторите попытку.
5) Вернитесь на исходную страницу и наслаждайтесь проектом )).
